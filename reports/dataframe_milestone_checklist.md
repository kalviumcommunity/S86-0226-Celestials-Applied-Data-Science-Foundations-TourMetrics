# Pandas DataFrame Fundamentals - Milestone Checklist

## Milestone Overview

This milestone focuses on creating and inspecting Pandas DataFrames from dictionaries and files.

## Learning Objectives

- [ ] Understand what a Pandas DataFrame represents
- [ ] Create DataFrames from Python dictionaries
- [ ] Load DataFrames from common file formats
- [ ] Inspect DataFrame structure and contents
- [ ] Recognize common issues during data loading

## Completion Checklist

### 1. Understanding Pandas DataFrames
- [ ] Understand rows and columns concept
- [ ] Recognize column names and indexes
- [ ] Compare DataFrames to spreadsheets
- [ ] Relate DataFrames to real-world tables

### 2. Creating DataFrames from Dictionaries
- [ ] Create a dictionary with column-style data
- [ ] Convert dictionary to DataFrame using `pd.DataFrame()`
- [ ] Inspect column names with `.columns`
- [ ] View data with `print()` or `.head()`
- [ ] Use at least 2 different examples

### 3. Creating DataFrames from Files
- [ ] Create or obtain a CSV file
- [ ] Load DataFrame using `pd.read_csv()`
- [ ] Understand how headers are interpreted
- [ ] Inspect the loaded data
- [ ] Recognize common parameters (sep, header, encoding)

### 4. Inspecting DataFrame Structure
- [ ] Use `.head()` to view first rows
- [ ] Use `.tail()` to view last rows
- [ ] Check shape with `.shape`
- [ ] List column names with `.columns`
- [ ] Check data types with `.dtypes`
- [ ] Use `.info()` for comprehensive overview
- [ ] Use `.describe()` for statistics

### 5. Video Walkthrough (~2 Minutes)
- [ ] Record screen-capture video
- [ ] Demonstrate creating DataFrame from dictionary
- [ ] Demonstrate loading DataFrame from file
- [ ] Show inspection methods (head, shape, dtypes)
- [ ] Explain why DataFrames are useful
- [ ] Keep video approximately 2 minutes
- [ ] Ensure screen is clearly visible
- [ ] Ensure audio is clear

## Common Mistakes to Avoid

- [ ] Not checking data after loading
- [ ] Using absolute file paths instead of relative
- [ ] Forgetting to import pandas
- [ ] Not handling file encoding issues
- [ ] Assuming all columns are same data type
- [ ] Not inspecting shape before analysis
- [ ] Confusing Series with DataFrame
